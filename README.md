arm64_mask_gen_py

PyO3 wrapper around `arm64-mask-gen` to expose pattern:mask generation to Python (used by the IDA plugin).

Build (recommended with maturin):

1. Install maturin:

   pip install maturin

2. Build & install into your current Python:

   maturin develop --release
